Annnnyways

This project is a keyboard that has the potential to have many different modules attached, modules such as displays, a rotary encoder panel, potentiometers, or a numpad!
> [!NOTE]
> Most of the features can be added due to modularity, but will not be developed by me personally

## Communication interface
The core concept of modularity comes down to 1 main factor, and that is the compatibility of the communication interface.
I2c is chosen for its ability to have multiple devices share the same signal lines, high support by many microcontrollers, and a decently high signal rate for information
